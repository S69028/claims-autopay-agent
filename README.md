# claims-autopay-agent
AX온보딩 교육 실습 - 자동심사 현황분석 Agent

자동심사 현황분석 Agent는 보험금 지급심사 워크플로우에서 자동지급 현황을 운영관리자 관점에서 빠르게 파악하고, 월말 스냅샷 기준의 전월 대비 변화와 segment 변화를 설명하기 위한 프로젝트입니다.

## 주요 기능
- 운영모니터링 화면 제공
- 리포트보관함 제공
- 월간 리포트 자동 생성 및 자동 발송
- 월간 리포트 재조회 및 재다운로드
- 조회기간 필터 제공
- 자동지급률, 처리효율, 안정 상태 표시
- 월말 기준 전월 대비 변화 및 segment 변화 설명
